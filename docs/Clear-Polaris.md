# Clear-Polaris

## SYNOPSIS
Clears the internal instance of Polaris

## SYNTAX

```
Clear-Polaris [<CommonParameters>]
```

## DESCRIPTION
Clears the internal Polaris .NET Standard object. 
The instance will be reinstantiated in other module calls.

## EXAMPLES

### EXAMPLE 1
```
Clear-Polaris
```
